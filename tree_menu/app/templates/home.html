    Created a "home_view" view that displays the Home page. Created an "about_view" that displays the About page. Created a "contacts_view" that displays the "Contacts" page.

  6. Templates:

    The templates "home.html", "about.html", and "contacts.html" were created in the "templates/app" folder.
